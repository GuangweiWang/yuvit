YUVIT

An open source convertor to YUV colorspace

What is YUVIT?

YUVIT is open source command line tool that can convert images into YUV format. This tool has some useful features:

  • Output format of yuv images: yuv, yuyv, yuv420, yuv444, planar, swapped and etc
  • Input format of images: png, jpeg, gif, bmp, tiff, tga and even camera raw images. For full list, please refer to FreeImage library page, which provides image support for yuvit.
  • It can write file sequenses or streams
  • It's crossplatform and runs on Mac, Windows and Linux

Usage

    yuvit [options] [-f format] [-s uvscale] <InFile> [OutFile]

    Options:
            -a : Add new image to the end of output file. Don't truncate output file.
            -m <start>:<end> : Multiple file input. Where:
                start : Sequence start
                end  : Sequence end
            -i : Interleave UV rows for planar formats
            -w : Swap UV components order

    Formats (-f option):
            yuv  : Planar format [DEFAULT]
            yuyv : Packed format
            uyvy : Packed format

    UV scales (-s option. Used only with -f and planar formats):
            h1v1 : UV not scaled down [DEFAULT]
            h2v2 : UV scaled down by 2x horizontally and vertically
            h2v1 : UV scaled down by 2x horizontally
            h1v2 : UV scaled down by 2x vertically

    Note: Use symbol '#' in file names for enumerators.

    Examples:
        yuvit -a -m 0:100 test###.bmp out.yuv
                Convert images from 'test000.bmp' to 'test100.bmp' into single 'out.yuv' file

        yuvit -m 10:200 test######.jpg out###.yuv
                Convert images 'test000010.jpg'...'test000200.jpg' into files 'out010.yuv'...'out200.yuv'

Installing

Windows

Simply unzip and run. There is no magic.

Mac OS

Simply unzip and run. There is also no magic.

Linux

Install it usng dpkg: 

sudo dpkg -i yuvit_0.1-1_i386.deb
Run.

History notes

I developed this tool for my main project in far 2005, but when I finished projet, I released tool as "any2yuv" on sourceforge. As far as tool was developed only for personal purposes, so it had some mistakes and sometime didn't work well. For years I didn't maintained this tool until in 2011 I got several requests and bug reports about it. After this I decided to rewrite this tool from scratch and put it on github with well organized source control. Enjoy it
